Super-Super Ejercicio

Crear un sitio web utilizando HTML Y CSS compuesto por, como mínimo, 4 páginas diferentes: Inicio, Servicios, Sobre nosotros y Contacto. La web irá sobre una temática elegida por vosotros. La web tendrá que tener un contenido coherente y relacionado conforme a la temática que elijáis, no vale con poner textos o contenido como placeholders (lorem ipsum, imágenes al azar, etc).

Deberán contener un menú principal con las distintas opciones de menú para permitir navegar al usuario a través de todo las páginas web, una sección donde se muestre el contenido principal de cada página, una barra lateral donde se muestren las últimas noticias de la página web (3 noticias con título imagen y breve descripción) y un pie que contenga 3 bloques diferentes. El primero contendrá un menú con las mismas opciones del menú principal, el segundo links a las distintas redes sociales de la página con la imagen de cada una de ellas y el tercer bloque los datos de contacto: dirección, telefonos, emails, etc… Esta estructura será común para todas las páginas.

Además de esto, cada página tendrá que tener su propio contenido obligatorio:

  • Todas las páginas tendrán como mínimo un encabezado.
  • La página Inicio describirá la temática de la web. Como mínimo deberá tener dos imágenes del ancho del div contenido, y dos párrafos explicando de qué va la página.
  • En la página de Servicios se deberán mostrar al menos tres servicios de la temática de la web (os podéis basar en la imagen dentro de la web). Deberás añadir una tabla de pricings (ejemplos) en referencia al tema de la web.
  • La página Sobre nosotros se deja a libre diseño del alumno.
  • La página Contacto tendrá como mínimo una imagen y dos formulario. Uno será un formulario de contacto (investigar qué llevan los formularios de ese tipo), con los inputs y los elementos correspondientes. El otro deberá ser un form de una newsletter (investigarlo también).

Puntos a tener en cuenta:

  • Crear primero la estructura del documento HTML y posteriormente añade los estilos necesarios par obtener el diseño deseado
  • Utilizar una hoja de estilos externa.
  • Para la página activa la opción de menú deberá resaltar de alguna forma para indicar al usuario en qué página se encuentra. Utiliza una lista (ul) para crear el menú.
  • Utilizar la etiqueta <div> para los distintos bloques y establecer un id para cada uno de ellos.

Guía para crear la estructura:

  • <div id="header"></div> → div que contendrá la cabecera de nuestra página web.
  • <div id="main"></div> → div que contendrá el contenido de cada página.
  • <div id="aside"></div> → div que contendrá la sección de noticias de nuestra página.
  • <div id="footer"></div> → div que contendrá el pie de nuestra página.

Utilizar en la medida de lo posible la propiedad float para la disposición de las capas y la propiedad clear

Almacenar las imágenes de nuestra web en una carpeta llamada images
